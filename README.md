## Key binding

> tmux - vim key binding

```
<c-a> + |      # split pane vertically
<c-a> + -      # split pane horizontally
<c-a> + c      # create new window
<c-a> + number # switching between window
<c-a> + [      # vim key binding, h,j,k,l,v,y etc. q to quit
```
